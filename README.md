# Study-space-booking-system-analysis
This project critically analyzes my university's study space booking system using BPMN, DFD, and ER diagrams.

## Overview
This project analyzes and redesigns Maastricht university's study space booking system used by students for both individual study spot reservations and group room bookings. Using systems analysis methodologies including DFD, BPMN, and ER diagram, the project evaluates the existing workflow and proposes an improved system structure for better efficiency.

This project was adapted and refined from a collaborative academic project.
